Task: Create a new text storage table.

Click on the "Add Table" button. When prompted, select the model you want to run on the text sent to the table and whether you want keyterms extracted by GPT-4 (requires giving us an OpenAI token).

The endpoint for sending the text and token for authentication will be shown once the table is created.

Using the endpoint will look like this:

curl -X POST \ -H "Content-Type: application/json" \ -d '{"text":"There was a sudden knock at the door, then silence."}' \ https://ai.featurebase.com/ingest/<endpoint_id>?token=<token>

One the job is complete, a sample response will look like this:

{ "_id": "FladzA", "keyterms": ['door', 'silence', 'sudden', 'knock'], "text": "There was a sudden knock at the door, then silence.", "embedding": [0.05333335, 0.020197952, 0.012141972, -0.041954238, -0.055371083, ...] }
